🚀 JavaScript Practice Projects

This repository contains multiple JavaScript practice questions designed to strengthen core programming logic.

Each question focuses on real-world problem solving using:

if-else conditions
loops (for, forEach)
arrays & objects
string methods
DOM manipulation
user input handling (prompt, alert)

Username & password validation

3. 🛒 Product Billing System
Search product from array

Apply discount based on price
DOM-based bill generation
4. ⏰ Employee Attendance System
Classifies attendance:
Full day

Absent
Overtime detection
5. 🔑 Password Strength Checker
Checks weak, medium, strong password

6. 💰 Bank Deposit System
Validates deposit amount
Detects large deposits
Updates account balance
7. 📅 Student Attendance Percentage
Calculates attendance %

8. 📱 Mobile Brand Analyzer
Identifies mobile category:
High-End
Mid-Range
Budget
9. 🧠 Quiz System
Simple math quiz
Score tracking
Final performance result
10. 🧾 Discount Billing System
Calculates total bill
